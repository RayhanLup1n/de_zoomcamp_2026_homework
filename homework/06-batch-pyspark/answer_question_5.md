# Question 5

## Task Explanation

Spark provides a User Interface (UI) for monitoring jobs, stages, tasks, and storage. This question asks for the default port of the Spark UI.

### Query Function

No query required. This is knowledge about Spark.

**Multiple Choice Options:**
- 80
- 443
- 4040
- 8080

**Answer:** **4040**

**Reason:** Spark UI runs by default on **port 4040**. Access via browser at `http://localhost:4040`.

**Other Spark Ports:**
| Port | Purpose |
|------|---------|
| 4040 | Spark UI (application) |
| 7077 | Spark Master (standalone cluster) |
| 8080 | Cluster Manager UI |
| 8081 | Worker UI |
